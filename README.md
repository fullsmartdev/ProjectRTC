# ProjectRTC

## WebRTC Live Streaming

- Node.js server
- Desktop client
- [Android client](https://github.com/pchab/AndroidRTC)

The signaling part is done with [socket.io](socket.io).
The client follows a MVVM pattern [knockoutjs](http://knockoutjs.com/)

## Install

- npm install
- npm start
